# abap_proyects
## Overview
This project is an ABAP report developed as part of a learning exercise. It processes purchasing documents from SAP standard tables (EKKO, EKPO, LFA1, ADR6) and generates structured output with calculated and validated information.

## Features
- Reads purchasing document data from SAP tables
- Calculates tax-adjusted amounts
- Counts and analyzes document positions
- Classifies approval levels based on business rules
- Validates supplier email addresses
- Displays output in two formats:
  - Classic WRITE report
  - ALV Grid display
- Logs each execution into a custom Z table

## Technologies
- ABAP (SAP ERP)
- ALV Reporting
- SAP Standard Tables (EKKO, EKPO, LFA1, ADR6)

## Purpose
This project was created to practice ABAP development, data processing logic, and report generation in SAP systems.
